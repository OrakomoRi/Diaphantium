What is Diaphantium?

Professional Automation Tool

Built for Tanki Online Players

Diaphantium is a free, open-source userscript that enhances your Tanki Online experience with automated actions, customizable hotkeys, and efficient supply management.

Why Diaphantium?

This project demonstrates advanced front-end development skills including CSS animations, JavaScript programming, cross-browser compatibility, and user experience design.
