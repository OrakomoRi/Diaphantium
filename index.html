What is Diaphantium?

Professional Automation Tool

Built for Tanki Online Players

Diaphantium is a free, open-source userscript that enhances your Tanki Online experience with automated actions, customizable hotkeys, and efficient supply management. Born from the idea of creating a simple popup interface for continuous key input, it has evolved into a comprehensive automation solution.

Why Diaphantium?

This project demonstrates advanced front-end development skills including CSS animations, JavaScript programming, cross-browser compatibility, and user experience design. It showcases the ability to reverse-engineer existing interfaces and create comprehensive visual overhauls while maintaining functionality.

Evolution Through Versions

May 2023

Version 1.0 - The Beginning

The initial release introduced a basic popup window with essential auto-clicking functionality. The interface was simple but functional, laying the foundation for future improvements.

June 2023

Version 2.0 - Dual Interface

Added two-tab system with Main and Settings tabs. Improved window dragging mechanics - users could now move the popup by any part except input fields. Enhanced visual design with better color schemes.

August 2023

Version 3.0 - Pro Features

Major overhaul with over 150 commits. Introduced navigation buttons, battle selection system, and advanced features like nickname blur and skip login. Added support for both team battles and deathmatch modes.

March 2024

Version 4.0 - Modern Era

Complete redesign with mobile support, dynamic hotkey management, and improved UI. Added ability to remove hotkeys, duplicate hotkey detection with visual warnings, and comprehensive changelog tracking.
